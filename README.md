# Fusion Fiesta Esports 🎮

Fusion Fiesta Esports is a modern BGMI tournament registration web application built with Flask and SQLite.

The platform allows players to register their squads, submit payment proofs, and participate in the tournament.

---

## 🚀 Features

- BGMI Squad Registration
- UPI / QR Code Payment Submission
- Payment Screenshot Upload
- Admin Dashboard
- Delete Registrations
- Export Player Data as CSV
- Countdown Timer for Tournament
- Modern Claymorphism Esports UI

## ⚙️ How to Run Locally

1️⃣ Clone the repository
git clone https://github.com/yourusername/fusion-fiesta-esports.git

2️⃣ Navigate to the project
cd fusion-fiesta-esports

3️⃣ Activate virtual environment
.\venv\Scripts\activate

4️⃣ Run the application
python app.py

5️⃣ Open in browser
http://127.0.0.1:5000
